==> Installation

1- Copier les fichiers sur votre serveur web ( Xampp,Wampp, ... )
2- if you are working on Linux machine verify that you have the
permitions to add or modify on this directory, if you do not have
permition, make sure to execute this command :
--> sudo chmod -R 777 covoiturage
3- Supprimer le fichier "includes/basededonnees.php" [ s'il existe ]
4- Lancer le fichier "initialise.php"
5- Renseignez les informations demandées (mot de passe : admin)
